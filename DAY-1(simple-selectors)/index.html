Types of selectors:
  • Simple selector:
    1. Element selector

      2. - (p), Selects all elements of a given type.

    2. Class selector

      3. - (.), Selects elements with a specific class.

    3. ID selector

      4. - (#), Selects a unique element with an ID.

    4. Group Selector

      5. - (,), Applies same style to multiple selectors.

    5. Universal Selector

      6. - (*),Selects all elements on the page.

      Priorities:
      1. id selector
      2. class selector
      3. elements, group selector
